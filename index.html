Build a Project Portfolio

If you ask most devs, they'll say that a strong set of public projects is a better marker for skills than a degree or certification. With project oriented courses, you'll learn through practical experience and end with your own personal to site to show off what you've built.

Neurodivergent Friendly

As an ADHDer myself, I know the strengths and challenges that come with neurodivergence. Using a more personalized approach, I’ll teach to your learning style with lessons made to work how you do along with your management tools, at your own pace.

Learn Popular Programming Languages

Javascript and CSS drive the web frontend and Python is used everywhere from backend web services to data science and lab research. They're two of the most popular languages, and knowing both will give you the flexibility to find more opportunities than focusing on a single area.

Interview Confidently

Frankly, the tech hiring process can really suck. Navigating it is a skillset separate from any coding experience that requires its own practice. Learn techniques for time-pressured coding challenges, how to talk the talk in interviews, and feel more confident applying to jobs.

Hack Like a Pro

There's a lot more to software development than learning languages and building apps. Getting comfortable with tools devs use like the command line console and code editors will make writing code easier and more fun, while knowing how networking and operating systems work will help you understand software at a deeper level.

Guided Self-Learning

Getting stuck on a problem and not knowing the answer happens all the time in software development. It's an open secret that knowing how to Google stuff and find the information you need is part of the job. With courses that include self-learning, you'll practice this kind of searching while building problem solving skills.

There's a lot to learn with software development, so courses are broken down into smaller, focused blocks. Shorter blocks help make progress feel faster with more consistent positive feedback. Since each block includes a project component, you'll also gain practical experience building as you go.

Though the blocks build on each other, they don't have to be done in the same order or can be skipped altogether. If there's a only few things you're interested in that's totally cool - you can focus on just what you want.
